# 🎮 睿玩智库

> 基于 FastAPI + React + 大语言模型的智能游戏问答与推荐系统


## 📋 项目概述

智能游戏对话系统是一个集成了多种AI技术的现代化Web应用，为游戏玩家提供智能问答、游戏推荐、文档问答和收藏管理等功能。系统采用前后端分离架构，支持多用户独立数据管理和实时流式对话。

### 🎯 核心功能

| 功能模块 | 描述 | 技术实现 |
|---------|------|---------|
| **🤖 智能对话** | 多模式AI对话，支持通用助手、游戏推荐、攻略查询等 | LangChain + 通义千问API |
| **📄 文档问答** | 上传PDF/Word文档，基于RAG技术进行智能问答 | ChromaDB + 向量检索 |
| **🎮 游戏推荐** | 个性化游戏推荐和攻略生成 | 智能分析 + Prompt工程 |
| **📚 收藏管理** | 用户独立的游戏收藏系统，支持分类和统计 | 本地存储 + 用户隔离 |
| **💬 聊天记录** | 按用户和功能分类的聊天历史管理 | 用户独立存储 |
| **🔐 用户系统** | 完整的用户注册登录和会话管理 | SQLite + PBKDF2加密 |

### 🌟 核心特性

#### 🔒 用户数据隔离
- **独立聊天记录**：每个用户拥有独立的聊天历史
- **个人游戏收藏**：用户专属的游戏收藏和管理
- **安全认证**：PBKDF2密码加密 + 会话管理
- **自动切换**：用户登录/退出时自动切换数据

#### 🤖 智能对话系统
- **多功能模式**：通用助手、游戏推荐、攻略查询、文档问答、游戏百科
- **流式响应**：实时消息流式传输
- **记忆管理**：按用户和功能分类的对话记忆
- **智能问题推荐**：基于用户收藏生成个性化问题

#### 📄 文档问答 (RAG)
- **多格式支持**：PDF、Word、TXT文档上传
- **智能分块**：文档自动分割和向量化
- **语义检索**：基于ChromaDB的向量相似度搜索
- **上下文问答**：结合文档内容的精准回答

#### 🎮 游戏管理系统
- **游戏收藏**：添加、删除、编辑游戏信息
- **分类管理**：按类型、平台、状态分类
- **统计分析**：收藏数据统计和可视化
- **智能推荐**：基于收藏生成问题推荐

## 🔨 开发指南

### 项目结构说明
- **main.py**：FastAPI应用主入口，定义所有API端点
- **llm_chain.py**：AI对话链管理，处理不同功能的对话逻辑
- **auth.py**：用户认证管理，处理登录注册和会话
- **database.py**：数据库操作，用户数据的CRUD操作
- **document_processing.py**：文档处理模块，支持RAG问答
- **models.py**：Pydantic数据模型，API请求响应格式定义

### 添加新功能
1. **后端**：在相应模块添加逻辑，在main.py添加API端点
2. **前端**：创建新组件，更新路由和状态管理
3. **数据**：更新数据模型和存储逻辑

### 代码规范
- **Python**：遵循PEP 8，使用类型注解
- **JavaScript**：使用ESLint，遵循React最佳实践
- **注释**：中文注释，详细说明功能和使用方法
